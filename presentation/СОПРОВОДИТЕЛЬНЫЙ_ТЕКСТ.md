### О системе

«СПС Голосовая Студия» — современное решение для создания качественных голосовых объявлений с использованием технологий искусственного интеллекта. Система позволяет оперативно генерировать аудиосообщения в профессиональном стандарте АИ-22, принятом в сфере транспорта.

---

### Ключевые возможности

| Функция | Описание |
|---------|----------|
| 🎤 **Синтез речи** | Генерация качественного аудио из текста |
| 🎭 **Эмоции** | Нейтральный, весёлый, грустный, серьёзный стили |
| 👥 **8 голосов** | Мужские и женские голоса с разными характеристиками |
| 📊 **Стандарт АИ-22** | Профессиональное качество звука (MP3, 44100 Гц, LUFS -23) |
| 📱 **PWA** | Установка на устройство как приложение |
| 📤 **Telegram** | Автоматическая отправка анонсов в каналы |
| 🔒 **Безопасность** | Защита паролем и управление сессиями |

---

### Область применения

- 🚇 Голосовые объявления на станциях метро
- 📢 Информирование пассажиров об изменениях в движении
- 📣 Анонсы культурных и спортивных мероприятий
- ⚠️ Экстренные оповещения
- 📻 Рекламные и информационные сообщения

---

### Технические требования

- **Доступ:** через веб-браузер (Chrome, Firefox, Safari, Edge)
- **Установка:** не требуется (работает в браузере)
- **PWA:** возможность установки на рабочий стол
- **Интернет:** требуется стабильное соединение

---

### Стандарт качества АИ-22

Все генерируемые аудиофайлы соответствуют профессиональному стандарту:

| Параметр | Значение |
|----------|----------|
| Формат | MP3 |
| Частота | 44100 Гц |
| Битрейт | CBR 128 кбит/с |
| Кодек | LAME |
| Каналы | Моно |
| LUFS | -23 ±0.5 LU |
| True Peak | ≤ -1 dBTP |

## Краткое описание для рассылки

**СПС Голосовая Студия** — веб-приложение для создания голосовых объявлений с использованием ИИ-синтеза речи. Позволяет генерировать профессиональные аудиосообщения в стандарте АИ-22 с выбором из 8 голосов и различных эмоциональных оттенков. Интеграция с Telegram для автоматической отправки анонсов. Разработано для Службы пассажирских сервисов ГУП «Петербургский метрополитен».
