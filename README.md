# 🚇 СПС Голосовая Студия

**Служба пассажирских сервисов • ГУП «Петербургский метрополитен»**

Система синтеза речевых объявлений с соблюдением вещательного стандарта АИ-22.

## ✨ Возможности

- 🎤 **7 профессиональных голосов** — Мужские и женские нейронные голоса
- 🎭 **9 интонаций** — Нейтральный, весёлый, грустный, серьёзный и другие
- 📻 **Стандарт АИ-22** — Broadcast-качество звука для громкой связи
- 📱 **PWA приложение** — Устанавливается как приложение
- 🔐 **Защита паролем** — Безопасный доступ к системе
- 📜 **История генераций** — Сохранение всех объявлений
- 📨 **Telegram интеграция** — Мгновенная отправка аудио

## 🚀 Установка и запуск

### Требования

- Node.js 18+
- Bun
- Python 3.10+
- FFmpeg

### Быстрый старт

```bash
# Установить зависимости
bun install

# Создать Python виртуальное окружение
python3 -m venv tts-env
source tts-env/bin/activate

# Установить Python зависимости
pip install edge-tts pydub numpy pyloudnorm soundfile scipy

# Инициализировать базу данных
bun run db:push

# Запустить TTS сервис (терминал 1)
cd mini-services/tts-service
source ../../tts-env/bin/activate
python3 index.py

# Запустить Next.js (терминал 2)
bun run dev
```

### Переменные окружения

Создайте файл `.env.local`:

```env
APP_PASSWORD=ваш-пароль
SESSION_SECRET=случайный-секретный-ключ
```
